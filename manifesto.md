# Pic2Scene
The desire of this project is to eventually get to the point where I can feed in any 2d image and I will get back a 3d stl file that I can use for my 3d printer.

4/30/2026 update: I want to use images with multiple angles to create the 3d model. If I can get this working well I may look to generative AI methods to fill gaps / improve these models.
